# 📊 Call Analytics Dashboard | Power BI

An interactive Power BI dashboard designed to analyze call records including incoming, outgoing, missed calls and spam detection.

## 🚀 Project Overview

This project analyzes phone call logs to identify call patterns, spam calls, call duration trends and category insights.

The dashboard helps in understanding communication behavior and detecting unusual call activities.

## 📂 Dataset

The dataset contains simulated call log data generated using Python.

Fields included:

- Phone Number
- Call Type (Incoming / Outgoing / Missed)
- Call Duration
- Call Category (Personal / Work / Spam / Unknown)

## 📊 Dashboard Features

- Total Calls KPI
- Incoming Calls KPI
- Outgoing Calls KPI
- Missed Calls KPI
- Spam Calls KPI

### Visual Insights

- Call Type Distribution (Donut Chart)
- Calls by Category (Bar Chart)
- Daily Call Trend (Line Chart)
- Total Call Duration by Type (Column Chart)
- Call Log Table

### Interactive Filters

- Call Type Filter
- Category Filter
- Phone Number Search
- Call Duration Filter

## 🛠 Tools Used

- Power BI
- Python (for dataset generation)
- Pandas
- CSV Dataset
